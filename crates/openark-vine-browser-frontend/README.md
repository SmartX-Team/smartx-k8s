# OpenARK VINE Browser Frontend

A frontend developed for a friendly GUI for a modular, high-performance storage backend.

[Source code for the reference backend implementation is also provided.](../openark-vine-browser-backend/README.md)

## Getting Started

At least one complete backend must be running on port `8888`.
If you're unsure which one to run, please enter the following command:

```bash
just run-openark-vine-browser-backend
```

Then, in your development machine, please type this command below:

```bash
just run-openark-vine-browser-frontend
```
